Arrays & Strings Interview Questions

The foundation of coding interviews. Focus on Two Pointers, Sliding Window, and O(1) space optimizations.

Core Array Algorithms (20+ Questions)

1. Two Sum: Given an array and a target, return indices of the two numbers such that they add up to the target. Easy
Technique: Hash Map. Complexity: O(n) time. Follow-up: Solve without extra space if array is sorted.
2. Container With Most Water: Find two lines that together with the x-axis form a container that contains the most water. Medium
Technique: Two Pointers (Greedy approach). Complexity: O(n) time. Avoid the brute force O($n^2$) solution.
3. Maximum Subarray (Kadane's Algorithm). Easy
Technique: Dynamic Programming. Complexity: O(n) time. Must handle negative numbers correctly.
4. Next Permutation: Find the lexicographically next greater permutation of numbers. Medium
Technique: Array Manipulation (Two Pointers/Reversal). Key Steps: Find pivot, find swap, reverse suffix.

Sliding Window & String Manipulation (20+ Questions)

5. Longest Substring Without Repeating Characters. Medium
Technique: Sliding Window (Hash Set/Map). Complexity: O(n) time.
6. Minimum Window Substring. Hard
Technique: Advanced Sliding Window with two Hash Maps. Used frequently in FAANG coding rounds.
7. Group Anagrams: Given an array of strings, group anagrams together. Medium
Technique: Hash Map (Key is the sorted string or character count map).
