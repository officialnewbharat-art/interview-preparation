Core Array Algorithms

1. Two Sum: Given an array and a target, return indices of the two numbers such that they add up to the target. Easy

Answer & Explanation

Optimal Technique: Use a Hash Map (or Dictionary) to store the required complement and its index as you iterate through the array. This reduces the time complexity from O(n2) to O(n).

Steps:

  1. Initialize an empty hash map numMap.
  2. Iterate through the array. For each number num, calculate the complement = target - num.
  3. Check if complement exists in numMap. If it does, you've found the pair! Return [numMap[complement], current_index].
  4. If the complement is not found, add the current number and its index to the map: numMap[num] = current_index.

Complexity: Time: O(n). Space: O(n).

2. Container With Most Water: Find two lines that together with the x-axis form a container that contains the most water. Medium

Answer & Explanation

Optimal Technique: Two-Pointer approach. Start with pointers at the extreme ends of the array (widest container) and move the pointer corresponding to the shorter line inward.

Rationale: The area is determined by min(height[left], height[right]) * (right - left). Starting wide maximizes (right - left). To increase the area, you must increase the minimum height. Moving the shorter line gives a better chance of finding a taller line.

Complexity: Time: O(n). Space: O(1).

3. Sort an array of 0s, 1s, and 2s without using a sorting algorithm (Dutch National Flag). Medium

Answer & Explanation

Technique: Three-pointer partition. Maintain pointers low, mid, and high. The array is partitioned into three sections:

  • Elements before low are 0s.
  • Elements between low and mid are 1s.
  • Elements after high are 2s.

If arr[mid] == 0, swap with arr[low] and increment low and mid. If arr[mid] == 2, swap with arr[high] and decrement high (only high moves). If arr[mid] == 1, only increment mid.

Strings & Sliding Window

4. Longest Substring Without Repeating Characters. Medium

Answer & Explanation

Technique: Sliding Window. Use a Hash Set (or a Map to store index) to track characters currently in the window defined by two pointers, left and right.

Process: Expand the window by moving right. If s[right] is a duplicate, shrink the window by moving left forward until the duplicate is removed. Update the maximum length after every valid step.

Complexity: Time: O(n). Space: O(min(n, alphabet_size)).

5. Minimum Window Substring. (Example of a Hard Question) Hard

Answer & Explanation

Technique: Advanced Sliding Window. Use two frequency maps: one for the target string (T) and one for the current window (S). Track the number of characters in the window that match the required count (the formed variable).

Process: Expand the window (right pointer). When formed equals the size of the target map, you have a valid window. Contract the window (left pointer) to find the minimum length while maintaining validity. Update the global minimum length.

Complexity: Time: O(|S| + |T|). Space: O(alphabet_size).
